# Portfolio — Brooke Cunningham

Personal portfolio site built with React and Vite.

## Stack

- React
- Vite
- JavaScript

## Running locally

npm install
npm run dev

Runs on http://localhost:5173

## Adding projects

Projects are defined in the projects array near the top of src/App.jsx:

const projects = [
  {
    title: "Project name",
    description: "What it does",
    tech: "React, Node.js, MongoDB",
    github: "https://github.com/BrookeCunningham/repo",
    demo: "https://live-link.com"
  }
]
